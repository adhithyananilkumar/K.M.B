<h1>Kunjacha Missionary Bhavan - Ramapuram</h1>

A Laravel-based web application.

## Requirements
- PHP 8.1 
- Composer 2.x
- MySQL 8+ 

<h2>NB: You may use Laravelherd for easy use.</h2>

# Install backend deps
 composer install


# Ensure database exists(MySQL), then run migrations
php artisan migrate


# Run app
php artisan serve   
```
